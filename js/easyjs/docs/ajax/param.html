param( obj )

静态方法Define in : ajax.js

将 key/value 的名值对转换成 URI 格式的查询字符串。比如 { foo1 : 'bar1', foo2 : 'bar2' } 将转换成 foo1=bar1&foo2=bar2。

参数
  • obj
    Object|
    Array
    必需

    key/value 的名值对的对象或数组。

返回值
  • String

    转换后的查询字符串。

例子1

将 data1 转换成查询字符串。

var data1 = [ { name : "first", value : "Rick" }, 
    { name:"last", value:"Astley" }, 
    { name:"job", value:"Rock Star" } ];

alert( E.param(data1) );

运行

例子2

将 data2 转换成查询字符串。

var data2 = { width : '200px', height : '300px' };

alert( E.param(data2) );

运行
