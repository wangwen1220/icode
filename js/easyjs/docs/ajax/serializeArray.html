serializeArray()

原型方法Define in : ajax.js

将 form 表单中的数据转换成 key/value 名值对格式的数组。这样就可以很方便的用 ajax 来提交 form 表单中的数据了。

返回值
  • Array

    转换后的数组。

例子1

将 form1 中的表单数据转换成 key/value 名值对格式的数组。

<form id="form1">
    <input type="text" name="testInput1" value="1234" />
    <input type="text" name="testInput2" value="4567" />
    <input type="radio" name="hello" checked="checked" />
    <input type="radio" name="hello" />
</form>
alert( E( '#form1' ).serializeArray() );
