delay( time )

原型方法Define in : anim.js

在实现动画队列时用于设置动画元素在延迟指定的时间后再进行动画。

参数
  • time
    Number
    必需

    延迟时间，单位为 ms。

返回值
  • easyJS Object

    返回 easyJS Object 自身，便于链式调用。

例子1

对 div 元素设置一组动画队列。

box1
E( '#box1' ).anim({ to : { left : '570px' }, duration : 600 })
    .anim({ to : { backgroundColor : '#fb219a' }, duration : 600 })
    .delay( 1000 )
    .anim({ to : { left : '335px' }, duration : 600 })
    .slideUp()
    .slideDown()
    .anim({  to : { backgroundColor : '#09c' }, duration : 1200 })
    .delay( 500 )
    .anim({ to : { left : '0px' }, duration : 500 })
    .anim({ to : { backgroundColor : '#2c2c2c' }, duration : 800 });
