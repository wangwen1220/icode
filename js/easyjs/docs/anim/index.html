动画模块

Define in : anim.js

该模块提供了让 easyJS Object 中的 DOM 元素可以流畅实现动画效果的一些方法。

easy.js 的动画模块支持多种形式的动画，除了常见的动画效果还支持颜色值的动画，背景图片的动画。模块还集成了一些优雅的缓动效果，动画队列的实现自然也不在话下，甚至还支持反向动画。有了这些强力的动画基础功能的支持，可以很方便的组合出复杂的动画效果。

如果你想深入了解动画原理，可以看看javascript动画剖析。
