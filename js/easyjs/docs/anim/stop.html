stop( [ clear, end ] )

原型方法Define in : anim.js

用于停止动画或清除动画队列。

参数
  • clear
    Boolean
    可选

    停止动画时，是否清除动画队列，默认为 false。

  • end
    Boolean
    可选

    停止动画时，是否执行当前队列的最后一帧动画。默认为 flase。

返回值
  • easyJS Object

    返回 easyJS Object 自身，便于链式调用。

例子1

对 div 元素设置一组动画队列。分别尝试 stop 的用法。
