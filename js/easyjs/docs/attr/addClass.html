addClass( name )

原型方法Define in : attr.js

给匹配元素添加一个或多个 className，多个 className 用空格分开。如果要删除 className，请使用 removeClass。

参数
  • name
    String|
    Function
    必需

    一个或多个字符串。如果该参数为函数，那么添加 className 的规则将按照函数提供的规则来添加，该函数接受两个参数，第一个参数是匹配元素的索引值，第二个参数是匹配元素的原 className。

返回值
  • easyJS Object

    返回 easyJS Object 自身，便于链式调用。

例子1

给 div 元素添加两个 className。

box1
<div id="box1">box1</div>
E( '#box1' ).addClass( 'test1 test2' );
alert( E.query( '#box1' )[0].className );

运行

例子2

给 li 元素分别设置不同的 className，参数是自定义函数。

  • test1
  • test2
  • test3
  • test4
  • test5
  • test6
<ul id="demoList1">
    <li>test1</li>
    <li>test2</li>
    <li>test3</li>
    <li>test4</li>
    <li>test5</li>
    <li>test6</li>
</ul>
E( '#demoList1 li' ).addClass(function( i ){
    return 'item_' + ( i + 1 );
});
alert( E( '#demoList1' ).html() );

运行
