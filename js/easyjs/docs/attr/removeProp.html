removeProp( name )

原型方法Define in : attr.js

删除匹配元素一个或多个 property，多个参数请使用空格隔开。

参数
  • name
    String
    必需

    property 属性名。多个属性请用空格隔开。

返回值
  • easyJS Object

    返回 easyJS Object 自身，便于链式调用。

例子1

删除 div 元素自定义的 property。

box1
<div id="box1">box1</div>
E( '#box1' ).prop( 'uid', 123456 );
alert( E( '#box1' ).prop('uid') );
E( '#box1' ).removeProp( 'uid' );
alert( E( '#box1' ).prop('uid') );
