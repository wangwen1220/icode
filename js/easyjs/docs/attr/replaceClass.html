replaceClass( source, target )

原型方法Define in : attr.js

将匹配元素包含了 source 的 className，替换成 target。

参数
  • source
    String
    必需

    被替换的 className。

  • target
    String
    必需

    用于替换的 className。

返回值
  • easyJS Object

    返回 easyJS Object 自身，便于链式调用。

例子1

将 div 元素的 test1 的 className 替换成 test2。

box1
<div id="box1" class="test1">box1</div>
E( '#box1' ).replaceClass( 'test1', 'test2' );
alert( E.query('#box1')[0].className );
