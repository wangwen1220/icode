val( [ content ] )

原型方法Define in : attr.js

无 content 参数，返回第一个匹配元素的指定的 value 值。有 content 参数，则设置所有匹配元素指定的 value值。主要用于表单元素的 value 属性的读写操作。

参数
  • content
    String
    可选

    待设置的 value 值。

返回值
  • easyJS Object

    返回 easyJS Object 自身，便于链式调用。

例子1

获取 input 元素的 value 值。

<input type="text" value="123456" id="demoInput1" />
alert( E( '#demoInput1' ).val() );

运行

例子2

设置 input 元素的 value 值。

<input type="text" id="demoInput2" />
E( '#demoInput1' ).val( '456789' );

运行
