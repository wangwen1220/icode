scrollLeft( [ val ] )

原型方法Define in : css.js

无 val 参数，则返回第一个匹配元素相对滚动条左侧的偏移位置。有 val 参数，则设置所有匹配元素相对滚动条左侧的偏移位置。如果要对滚动条顶部的偏移位置进行读写操作，请使用 scrollTop。

参数
  • val
    Number
    可选

    相对于滚动条左侧的偏移位置值。

返回值
  • Number

    无 val 参数返回不带单位的数值。

  • easyJS Object

    有 val 参数返回 easyJS Object 自身，便于链式调用。

例子1

获取 div 元素相对于滚动条左侧的偏移位置值，可以尝试拖动 div 的滚动条再查看结果。

para1

<div id="box1" style="width:300px;overflow-x:auto">
    <p style="width:500px">para1</p>
</div>
alert( E('#box1').scrollLeft() );

运行

例子2

设置 div 元素相对于滚动条左侧的偏移位置值。

para2

<div id="box1" style="width:300px;overflow-x:auto">
    <p style="width:500px">para1</p>
</div>
E( '#box2' ).scrollLeft( 80 );

运行
