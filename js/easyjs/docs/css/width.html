width()

原型方法Define in : css.js

获取第一个匹配元素的实际宽度值，实际宽度值不包含 borderWidth、paddingLeft、paddingRight。如果想要获取宽度值，请使用 height。

返回值
  • Number

    不带单位的数值。

例子1

获取 div 元素的宽度。

box1
<div id="box1" style="width:300px;border:5px solid #ccc;padding:5px;">box1</div>
alert( E('#box1').width() );
