removeData( name )

原型方法Define in : data.js

删除该 DOM 元素的缓存数据。

参数
  • name
    String
    必需

    缓存数据的属性名。

返回值
  • easyJS Object

    返回 easyJS Object 方便链式调用。

例子1

将一个对象写入到 div 元素的缓存中。

HTML代码：
<div></div>
Javascript代码：
E( 'div' ).data( 'testData', { foo : 'bar' } );

删除该数据。

Javascript代码：
E( 'div' ).removeData( 'testData' );
console.log( E('div').data('testData') ); // undefined
