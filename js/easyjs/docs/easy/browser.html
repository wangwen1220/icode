browser

ObjectDefine in : easy.js

存储当前浏览器的型号和版本的对象，该对象只有这2个固定的属性，其中 version 属性是当前浏览器的版本号，其他浏览器型号请看下表。

属性名 对应浏览器
ie

Internet Explorer浏览器
chrome

Chrome(谷歌)浏览器
firefox

Firefox(火狐)浏览器
safari

Safari浏览器
opera

Opera(欧朋)浏览器
例子1

检测当前浏览器是否为 firefox，如果是则返回其版本号。

JavaScript代码：
alert( E.browser.firefox );
            
if( E.browser.firefox ){
    alert( E.browser.version );
}
