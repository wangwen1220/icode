config( options )

静态方法Define in : easy.js

用于配置模块加载器的初始化配置。查看模块加载器的详细说明。

参数
  • options
    Object
    必需
    • baseUrl

      模块加载时所依赖的全局基础路径，默认为 easy.js 加载时的 url。

    • charset

      模块对应的 charset 存储对象，默认不设置模块的 charset 参数。

例子1

修改模块加载器的 baseUrl。

JavaScript代码：
E.config({
    baseUrl : 'http://easyjs.org/js/'
});
例子2

修改 test1 模块的 charset 为 utf-8。

JavaScript代码：
E.config({
    charset : { test1 : 'utf-8' }
});
