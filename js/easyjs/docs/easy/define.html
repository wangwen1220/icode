define( name, [ deps ], factory )

全局方法Define in : easy.js

定义一个模块，该方法尽量遵循 AMD规范。查看模块加载器的详细说明。

参数
  • name
    String
    必需

    模块名，必须和该模块的文件名保持一致。

  • deps
    String|
    Array
    可选

    依赖模块的标识，单个模块可以是字符串形式的参数，多个模块必须为数组形式的参数。

  • factory
    Function
    必需

    模块内容。

例子1

定义名叫 hello 的模块。

JavaScript代码：
define( 'hello', function(){
    console.log( 'hello ' );
});

定义 world 模块，world 有 exports。

JavaScript代码：
define( 'world', function(){
    return 'world';
});

hello 模块依赖 world 模块。

JavaScript代码：
define( 'hello', 'world', function( world ){
    console.log( 'hello ' + world );  // 'hello world'
});

定义 hello 模块，有多个依赖模块。

JavaScript代码：
define( 'hello', [ 'test1', 'test2', 'test3' ], function(){
    console.log( 'hello ' );
});
