euid

StringDefine in : easy.js

在 easy.js 加载的时候会生成一个以 'easyJS_' 为前缀的 uuid，该 id 一旦生成是不会变化的，除非刷新页面，为了区别其他的 uuid，故名 euid。euid 在框架内部应用广泛，主要用于数据存储。

例子1

查看当前的 euid。

JavaScript代码：
alert( E.euid );
