guid( pre )

静态方法Define in : easy.js

生成一个唯一的uuid，在 easy.js 加载的时候会使用该方法生成一个 euid，euid 在框架内部应用广泛，主要用于数据存储。

参数
  • pre
    String
    可选

    uuid的前缀，默认的前缀是'easyJS_'。

返回值
  • String

    合并的新对象。

例子1

生成一个uuid。

JavaScript代码：
alert( E.guid() );
