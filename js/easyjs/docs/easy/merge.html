merge( Objects )

静态方法Define in : easy.js

将多个对象合并成一个新对象，后面的对象属性将覆盖前面的，常用于合并配置对象。

参数
  • Objects
    Object
    必需

    一个或多个待合并的对象。

返回值
  • Object

    合并的新对象。

例子1

将 a、b 合并成一个新的对象。

JavaScript代码：
var a = { name : 'test1' },
    b = { foo : 'bar', name : 'easy.js', value : 'test2' },
    c;

c = E.merge( a, b );
console.log( c ); // { name : 'easy.js', foo : 'bar', value : 'test2' }
