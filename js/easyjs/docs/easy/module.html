module

ObjectDefine in : easy.js

用于保存模块加载的信息，使用模块名作为键值，模块信息就保存在属性值中。查看模块加载器的详细说明。

API 数据类型 说明
deps Array

该模块的依赖列表。
exports Anything

模块的外部接口，可以是任意值。
status Number

模块的状态值，分为1、2、3、4这4个状态值。

1 模块开始加载。

2 模块加载完毕，开始解析模块。

3 模块解析完毕，所有的依赖模块也都加载完，但还未输出 exports。在未输出 exports 前，可以使用 modify 方法对 exports 进行修改。

4 模块加载并解析完毕，exports已可用。
url String

模块的加载url。
例子1

访问 hello 模块的模块信息。

JavaScript代码：
console.log( E.module.hello );
/*
 * status : 4,
 * url : 'http://example.com/hello.js' 
 */
