each( Obj, fn, [ context ] )

静态方法Define in : lang.js

遍历对象，并执行回调函数。该方法只能遍历 Object，不能遍历数组，如需遍历数组，请使用原生方法 forEach。

参数
  • Obj
    Object
    必需

    待遍历的对象。

  • fn
    Function
    必需

    遍历对象时执行的回调函数，如果回调函数返回 false，将跳出循环。回调函数接受2个参数，第一个参数是对象的属性名，第二个参数是对应的属性值。

  • context
    Object
    可选

    回调函数的 this 指向对象。

返回值
  • Object

    被遍历的对象。

例子1

遍历对象并执行回调。

JavaScript代码：
E.each( { foo : 'bar' }, function( name, val ){
    console.log( name );  // foo        
    console.log( val );   // bar             
});
