语言扩展模块

Define in : lang.js

JavaScript是一门很灵活的语言，对于原生 JavaScript 语言的扩展也是可浅可深。本模块暂时只保守的扩展了一些最基本的功能，比如数据类型判断，数据解析等功能。另外本模块又大胆的在原生对象的原型链上扩展了一些比较常见的 ECMAScript5 的方法，让开发者可以在不支持 ECMAScript5 的浏览器中放心的使用常见的新方法而不用担心兼容性问题。
