Array.prototype.indexOf( item, index )

原型方法Define in : lang-patch.js

查找某数组元素在数组中的索引，如果不包含此元素则返回-1。indexOf 是从左到右的查找顺序，lastIndexOf 是从右到左的查找顺序。

参数
  • item
    Object
    必需

    要查找的数组元素。

  • index
    Number
    可选

    查找的起始索引，如果此参数为负数，则是数组末尾的偏移量。未传参，则从0索引位置开始查找。

返回值
  • Number

    匹配到数组元素的索引。

例子1

查找5、10在数组中的索引值。

JavaScript代码：
var index1 = [ 1, 3, 5, 7 ].indexOf( 5 );
console.log( index1 );  // 2
    
var index2 = [ 1, 3, 5, 7 ].indexOf( 10 );
console.log( index2 );  // -1
