isFunction( o )

静态方法Define in : lang.js

判断是否为函数。

参数
  • o
    Object
    必需

    任意对象。

返回值
  • Boolean

    是函数则返回 true，否则返回 false。

例子1

判断变量 test 是否为函数。

JavaScript代码：
var test = function(){};
console.log( E.isFunction(test) );  // true
