Object.keys( obj )

静态方法Define in : lang-patch.js

遍历对象，将对象的属性名组成数组返回( 不包含原型链上的属性名 )。

参数
  • obj
    Object
    必需

    待遍历的对象。

返回值
  • Array

    对象的属性名组成的数组。

例子1

将对象的属性名组成数组。

JavaScript代码：
var arr = Object.keys({ name : 'easy.js', msg : 'is very easy to use' });
console.log( arr ); // [ 'name', 'msg' ]
