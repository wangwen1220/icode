makeArray( source, target )

静态方法Define in : lang.js

将任意对象转换成数组。

参数
  • source
    Object
    必需

    任意对象。

  • target
    Array
    必需

    目标数组。

返回值
  • Array

    目标数组。

例子1

将任意对象转换成数组。

JavaScript代码：
console.log( E.makeArray('hello') );  // [ 'hello' ]
console.log( E.makeArray({ foo : 'bar' }) );  // [ { foo : 'bar' } ]
例子2

将任意对象添加到数组 a 中。

JavaScript代码：
var a = [ 'source' ];
console.log( E.makeArray('hello', a) );  // [ 'source', 'hello' ]
console.log( E.makeArray({ foo : 'bar' }, a) );  // [ 'source', 'hello', { foo : 'bar' } ]
console.log( E.makeArray([ 'test1' ], a) );  // [ 'source', 'hello', { foo : 'bar' }, 'test1' ]
