parseJSON( data )

静态方法Define in : lang.js

将字符串转换成 JSON 对象。

参数
  • data
    String
    必需

    JSON 字符串。

返回值
  • JSON

    JSON 数据。

例子1

将字符串转换成 JSON 对象。

JavaScript代码：
var jsonData = '[{"restore":false,"name":"foo"},{"restore":true,"name":"bar"}]';
console.log( E.parseJSON(jsonData)[0].name ); // 'foo'
