appendTo( content )

原型方法Define in : node.js

将匹配的元素追加到一个指定元素的内部末尾处。appendTo 是将自己追加到指定元素内，append 是将指定元素追加到自己的内部。如果要将匹配元素插入到指定元素中的起始位置处，请使用 prependTo。

参数
  • content
    htmlString|
    Element|
    Element Array|
    easyJS Object|
    Function
    必需

    content 可以是 HTML 字符串、DOM 元素、DOM 元素数组、easyJS Object。如果该参数是函数，那么插入规则将按照函数提供的规则来插入，该函数只接收一个索引参数，但是函数必须返回一个可以插入的数据类型。

返回值
  • easyJS Object

    返回 easyJS Object 便于链式调用。

例子1

将新建的 span 元素追加到一个 div 元素内部的末尾处，参数是 DOM 元素。

box1
<div id="box1">box1</div>
E( '<span> new span</span>' ).appendTo( E.query('#box1')[0] );

运行

例子2

将 em 元素追加到一个新建的 p 元素中，然后将该 p 元素追加到 div 元素内部的末尾处。

em1
box2
<em id="demoEm1">em1</em>
<div id="box2">box2</div>
E( '#demoEm1' ).appendTo( '<p/>' )
    .parent()
    .appendTo( E.query('#box2')[0] );

运行

例子3

将 span 元素追加到 a 元素内部的末尾处，参数是 easyJS Object。

span link 
<span id="demoSpan1">span </span><a href="#" id="demoLink1">link </a>
E( '#demoSpan1' ).appendTo( E('#demoLink1') );

运行

例子4

将 list1 中的 a 元素按顺序追加到 list2 中的 span 元素的内部末尾处，参数是自定义的函数。

list1
link1
link2
list2
span1
span2 
<dl id="demoList1">
    <dt>list1</dt>
    <dd><a href="#">link1</a></dd>
    <dd><a href="#">link2</a></dd>
</dl>
<dl id="demoList2">
    <dt>list2</dt>
    <dd><span>span1</span></dd>
    <dd><span>span2</span></dd>
</dl>
var spans = E.query( '#demoList2 span' );
E( '#demoList1 a' ).appendTo(function( index ){
    return spans[ index ];
});

运行
