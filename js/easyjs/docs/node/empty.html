empty()

原型方法Define in : node.js

清空匹配元素的子元素，并删除子元素存储在 E.cache 中的数据和卸载使用 E.prototype.on 绑定的事件。如果要删除匹配元素，并清空匹配元素的子元素，请使用 remove。

返回值
  • easyJS Object

    清空后的匹配元素。

例子1

清空 ul 元素的所有子元素，并卸载事件。

  • test1
  • test2
  • test3
  • test4
<ul id="demoList1">
    <li class="item1">test1</li>
    <li class="item2">test2</li>
    <li class="item3">test3</li>
    <li class="item4">test4</li>
</ul>
E( '#demoList1 .item1' ).on( 'click', function(){
    alert( this.innerHTML );
});

E( '#demoList1' ).empty();
