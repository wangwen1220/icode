first()

原型方法Define in : node.js

筛选索引为0的匹配元素，同 eq(0) 的结果一样。如果想筛选最后一个匹配元素，请使用 last。

返回值
  • easyJS Object

    第一个匹配元素。

例子1

查找 ul 元素的所有 li 元素，并筛选出第一个 li 元素，给该元素添加一个背景颜色。

  • test1
  • test2
  • test3
  • test4
<ul id="demoList1">
    <li class="item1">test1</li>
    <li class="item2">test2</li>
    <li class="item3">test3</li>
    <li class="item4">test4</li>
</ul>
E( '#demoList1 li' ).first().css( 'backgroundColor', '#ccc' );
