is( selector )

原型方法Define in : node.js

使用选择器来检测匹配元素是否符合选择器的匹配条件，符合则返回 true，否则返回 false。

参数
  • selector
    String
    必需

    用于检测的选择器。支持基本选择器、属性选择器、伪类选择器。

返回值
  • Boolean

    是否匹配。

例子1

检测 ul 元素中的第一个 li 元素的 class 是否为 item1。

  • test1
  • test2
  • test3
  • test4
<ul id="demoList1">
    <li class="item1">test1</li>
    <li class="item2">test2</li>
    <li class="item3">test3</li>
    <li class="item4">test4</li>
</ul>
alert( E( '#demoList1 li:first-child' ).is( '.item1' ) );
