map( fn )

原型方法Define in : node.js

遍历匹配元素，并执行回调，根据回调函数的返回值合并成一个新的数组。

参数
  • fn
    Function
    必需

    回调函数，该函数接受两个参数，第一个参数是当前遍历的元素索引值，第二个参数是 easyJS Object 自身。函数的 this 指向当前遍历的元素。

返回值
  • Array

    返回合并好的数组。

例子1

遍历 ul 元素中的所有 li 元素，将 li 元素的 innerHTML 合并成数组。

  • test1
  • test2
  • test3
  • test4
<ul id="demoList1">
    <li class="item1">test1</li>
    <li class="item2">test2</li>
    <li class="item3">test3</li>
    <li class="item4">test4</li>
</ul>
var arr = E( '#demoList1 li' ).map(function(){
    return this.innerHTML;
});

alert( arr );
