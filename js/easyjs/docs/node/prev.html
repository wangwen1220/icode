prev( [ selector ] )

原型方法Define in : node.js

查找匹配元素的前面的同辈元素，如果选择器不为空将遍历前面的同辈元素，直到查找到匹配的元素，只要查找到第一个符合条件的元素则立即返回，否则将只查找紧邻匹配元素的前面的同辈元素。如果要查找匹配元素后面的同辈元素，可以使用 next。

参数
  • selector
    String
    可选

    用于查找的选择器，支持基本选择器、属性选择器、伪类选择器。

返回值
  • easyJS Object

    查找结果。

例子1

查找第二个 li 元素前面的元素，并给元素添加一个背景颜色。

  • test1
  • test2
  • test3
  • test4
<ul id="demoList1">
    <li class="item1">test1</li>
    <li class="item2">test2</li>
    <li class="item3">test3</li>
    <li class="item4">test4</li>
</ul>
E( '#demoList1 li:nth-chlid(2)' ).prev().css( 'backgroundColor', '#ccc' );

运行

例子2

查找最后一个 li 元素前面的 class 为 .item 的同辈元素( 只会匹配到第一个.item )，并给元素添加一个背景颜色。

  • test1
  • test2
  • test3
  • test4
<ul id="demoList1">
    <li class="item">test1</li>
    <li class="item">test2</li>
    <li class="item3">test3</li>
    <li class="item4">test4</li>
</ul>
E( '#demoList2 li:last-child' ).prev( '.item' ).css( 'backgroundColor', '#ccc' );

运行
