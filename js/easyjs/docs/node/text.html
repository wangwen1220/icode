text( [ content ] )

原型方法Define in : node.js

当无 content 参数时，该方法会立即返回第一个匹配元素的 textContent 内容。有 content 参数时，则将 content 设置成所有匹配元素的 textContent。

参数
  • content
    String|
    Function
    可选

    需要设置的 textContent 字符串，如果该参数是函数，那么将按照函数提供的规则设置 textContent 内容，该函数接收两个参数，第一个参数是匹配元素的索引值，第二个参数是匹配元素的原 textContent 内容。

返回值
  • String

    无 content 参数则返回第一个匹配元素的 textContent 内容。

  • easyJS Object

    有 content 参数则返回 easyJS Object 自身，便于链式调用。

例子1

获取 div 元素的 textContent 内容。

para1

<div id="box1"><p>para1</p></div>
alert( E('#box1').text() );

运行

例子2

设置 div 元素的 textContent 内容，参数是字符串。

box2
<div id="box2">box2</div>
E( '#box2' ).text( 'para2' );

运行

例子3

设置 dd 元素的 textContent 内容，参数是函数。

list1
dd1
dd2
<dl id="demoList1">
    <dt>list1</dt>
    <dd>dd1</dd>
    <dd>dd2</dd>
</dl>
E( '#demoList1 dd' ).text(function( i, content ){
    return i + content;
});

运行
