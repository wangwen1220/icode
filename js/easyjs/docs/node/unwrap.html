unwrap()

原型方法Define in : node.js

将匹配元素移出父元素。

返回值
  • easyJS Object

    返回 easyJS Object 便于链式调用。

例子1

删除 p 的父级元素 div。

para1

<p id="para1">para1</p>
E( '#para1' ).unwrap();

运行
