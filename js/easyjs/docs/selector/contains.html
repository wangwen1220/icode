contains( a, b )

静态方法Define in : selector.js

检测 a 元素是否包含了 b 元素，如果包含则返回 true，否则返回 false。

参数
  • a
    Element
    必需

    包含元素。

  • b
    Element
    必需

    被包含元素。

返回值
  • Boolean

    是否包含。

例子1

检测 div 是否包含了 p。

<div><p></p></div>
var div = E.query( 'div' )[0],
    p = E.query( 'p' )[0];
    
console.log( E.contains(div, p) ) // true
