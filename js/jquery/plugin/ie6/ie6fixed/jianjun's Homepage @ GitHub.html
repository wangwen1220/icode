ie6fixed

说明

  • ie6fixed解决IE6不支持fixed定位的问题
  • ie6fixed提供两个版本，一个是普通版本，另一个是seajs module
  • ie6fixed依赖jQuery

API

  • fixed
  • @param {jquery object} obj 需要fixed定位的元素
  • @param {string / function} position 定位方式，position可以是cc lt rt lb rb字符串，也可以是自定义的定位函数
    cc:页面居中定位，lt:左上角定位，rt:右上角定位，lb:左下角定位，rb:右下角定位
