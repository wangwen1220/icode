    Thanks Collin!

  • Barry Jones

    Great plug-in thanks!  I needed to be able to programmatically start/stop the rumbling, so I added the following code at line 151:

    if(opt.rumbleEvent === 'custom'){ $obj.bind('start_rumble', function(){ var rumblee = $(this); rumbleInterval = setInterval(function() { rumbler(rumblee); }, rumbleSpeed); }); $obj.bind('stop_rumble', function(){ var rumblee = $(this); clearInterval(rumbleInterval); rumblee.css(resetRumblerCSS); rumblee.css(objXrel, objXmove+'px'); rumblee.css(objYrel, objYmove+'px'); if(inlineChange === true){ rumblee.css('display','inline'); } }); }

    Now to start rumble: 

    my_dom_element.jrumble({rumbleEvent: 'custom'});my_dom_element.trigger('start_rumble');

    .. and to stop:

    my_dom_element.trigger('stop_rumble');

    Hope this helps someone!

    Barry
