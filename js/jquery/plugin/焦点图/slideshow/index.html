jQuery Slide Show 焦点图轮换 v2

  • ·基于jQuery v1.4.4
  • ·支持4种轮换方式
  • ·支持自定义切换事件
  • ·支持自定义渐变速度
  • ·支持自定义轮换时间

注意事项：

1.内部会自动创建切换按钮元素，也可在外部直接创建，若外部已创建，内部将跳过此步骤；

2.展示图片未超过1张时，会隐藏切换按钮元素。

【settings 参数说明】（可选）

名称 默认值 说明
events "click" 触发按钮事件的方式
type "x" 轮换效果类型。可设置为："x"|"y"|"fade"|"toggle"
start 0 首次展示序号
speed 800 切换速度（单位：毫秒）
time 5000 自动轮播间隔时间（单位：毫秒）
auto true 是否自动轮播
btn true 是否使用切换按钮
plus false 是否使用 plus 按钮
minus false 是否使用 minus 按钮

【DOM 结构】

<!-- 焦点图 -->
<div id="element_id">
	<div class="box">
		<ul class="list">
			<li></li>
			<li></li>
			...
			<li></li>
		</ul>
	</div>

	<!-- 内部自动创建的切换按钮元素 开始 -->
	<ul class="btn">
		<li class="b_1">1</li>
		<li class="b_2">2</li>
		...
		<li class="b_n">n</li>
	</ul>
	<div class="plus"></div>
	<div class="minus"></div>
	<!-- 内部自动创建的切换按钮元素 结束 -->
</div>

【JavaScript 调用方式】

// 直接调用
$("#element_id").slideShow();

// 自定义参数调用
$("#element_id").slideShow({
	events:"click",
	type:"x",
	speed:800,
	time:5000,
	auto:true,
	btn:true,
	plus:false,
	minus:false
});
