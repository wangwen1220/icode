图片轮播

多张图片并列展示，可以采用横排或者竖排的方式，通过两边的按钮切换选中的图片，支持自动滚动和循环滚动，多用于新闻图片轮播和图片产品的图片轮播展示。

图片轮播插件

button

为图片轮播组件添加控制前后按钮插件

autoScroll

为图片轮播组件增加自动滚动功能

fx

为图片轮播组件增加动画滚动功能

参考手册

参数配置

  • options.items : Array
    描述每个滚动项的字符数据，格式：[{content: 'text-0'}, {content: 'text-2'}, {content: 'text-3'}...].
  • options.orientation : Number
    描述该组件是创建一个横向滚动组件或是竖向滚动组件，取值：{horizontal: 横向, vertical: 竖向}，默认horizontal
  • options.originalIndex : Number
    默认选项卡的聚焦项，默认0
  • options.viewSize : Number
    描述一页显示多少个滚动项，默认3
  • options.focusRange : Object
    描述焦点在viewSize参数中的滚动范围，区域起始位从0开始，格式：{min: 0, max: 4}，当焦点超出focusRange指定的范围时才会触发可视区域的滚动动作，默认{min: 0, max: options.viewSize - 1 || 2}
  • options.isLoop : Boolean
    是否支持循环滚动，默认false
  • options.step : Number
    描述每次调用focusPrev或focusNext方法时一次滚动过多少个项，默认1
  • options.fx.enable : Boolean
    插件开关，默认 true
  • options.fx.duration : Number
    执行一次动画的时间(毫秒)，默认500
  • options.fx.interval : Number
    动画脉冲间隔时间(毫秒)，默认16
  • options.button.enable : Boolean
    插件开关，默认true
  • options.autoScroll.enable : Boolean
    插件开关，默认true
  • options.autoScroll.interval : Number
    以毫秒描述每次滚动的时间间隔，默认1000
  • options.autoScroll.direction : String
    取值，forward|backward 描述组件的滚动方向，默认forward
  • options.button.buttonLabel : Object
    前后按钮的文字描述，参考值：{prev: 'left', next: 'right'}

组件方法

  • render

    根据参数指定的容器渲染滚动项

    • 参数

      • target: {String|HTMLElement}

        被渲染的指定容器

  • focusPrev

    以step为单位翻到上一项

  • focusNext

    以step为单位翻到下一项

  • focus

    根据方向从当前项聚焦到index指定的项

    • 参数

      • index: {Number}

        滚动项的索引

      • direction: {String}

        可选，滚动方向，取值：forward|backward

  • getCurrentIndex

    取得当前得到焦点项在所有数据项中的索引值

    • 返回值: {Number}

      索引值.

  • getTotalCount

    取得数据项的总数目

    • 返回值: {Number}

      总数.

  • start

    启动滚动

  • stop

    停止滚动

  • isFirst

    是否已经滚动到首项

    • 返回值: {Boolean}

      当已经滚动到首项时返回true，否则返回false

  • isLast

    是否已经滚动到末项

    • 返回值: {Boolean}

      当已经滚动到末项时返回true，否则返回false

  • getElement

    取得 ui 模块对应的 dom element 对象

    • 参数

      • key: {String}

        该ID对应的关键字(可选参数)

    • 返回值: {Object}

      ui模块对应的dom element

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合

组件事件

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

      • evt.index: {Number}

        取得触发时该滚动项的索引值

      • evt.DOMEvent: {Event}

        取得当时触发的浏览器事件对象

  • onmouseoveritem

    鼠标划入单个滚动项时触发

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

      • evt.index: {Number}

        取得触发时该滚动项的索引值

      • evt.DOMEvent: {Event}

        取得当时触发的浏览器事件对象

  • onmouseoutitem

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

      • evt.index: {Number}

        取得触发时该滚动项的索引值

      • evt.DOMEvent: {Event}

        取得当时触发的浏览器事件对象
