组合框

在原生select下拉控件上增加可编辑选中内容的功能，该组件由一个文本输入控件和一个下拉菜单组成，多用于替代浏览器原生的select下拉组件，支持下拉菜单的内容根据输入的内容自动变化，类似suggestion输入框提示。

组合框插件

suggestion

组合框的输入框提示功能插件

参考手册

参数配置

  • options.items : Array<Object>
    组合框下拉菜单的数据，每项由value和content组成，默认[]。
  • options.viewSize : Number
    下拉菜单最多显示的项目数，若选项多于此配置，则出现纵向滚动条，默认5。
  • options.readonly : Boolean
    输入框是否可以编辑输入，默认true。
  • options.disabled : Boolean
    组合框是否处于禁用状态，默认false。
  • options.originIndex : Number
    初始化后默认选中的值的索引，不选中任何项为-1，当readonly为true时，默认0，反之默认-1。
  • options.width : Number|String
    组合框的宽度，默认100%。
  • options.suggestion.enable : Boolean
    插件开关，默认true

组件方法

  • render

    将组合框渲染到dom中

    • 参数

      • target: {String|HTMLElement}

        渲染的容器，默认为document.body。

      • position: {String}

        渲染的位置，可以是beforeBegin, afterBegin, beforeEnd, afterEnd，默认beforeEnd。

  • getValue

    获得选中项的值，若输入框的中的值不在下拉菜单中，则返回输入框中的值。

    • 返回值: {Number}

      ComboBox的值

  • getSelectIndex

    获得选中项的索引值，若输入框的中的值不在下拉菜单中，则返回-1.

    • 返回值: {Number}

      索引值

  • setByValue

    根据选中项的值设置组合框

    • 参数

      • value: {String|Number|Boolean}

        选中项的值。

  • focus

    使组合框获得焦点

  • blur

    使组合框失去焦点

  • reset

    将组合框重置

  • disable

    设置组合框为不可用状态。

  • enable

    设置组合框为可用状态

  • setWidth

    设置组合框的宽度

    • 参数

      • width: {Number|String}

        宽度数字

  • getElement

    获得 组合框组件结构里的 HtmlElement对象

    • 参数

      • name: {String}

        可选的值包括：container(combobox节点)|input-container(输入框和右边的箭头整体)|input(输入框)|arrow(输入框右边的箭头)|menu(下拉菜单)

    • 返回值: {HtmlElement}

      得到的 HtmlElement 对象

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合

  • onchange

    组合框值变化时触发。有两个途径：1，confirm后若值发生变化；2，blur后若值发生变化。

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

      • evt.from: {Number}

        如何触发该事件，可以为comfirm或blur。

      • evt.result.index: {Number}

        选项的索引（仅在evt.from为confirm时）

      • evt.result.value: {Number|Boolean|String}

        选项的值（仅在evt.from为confirm时）

      • evt.result.content: {String}

        选项的文字 （仅在evt.from为confirm时）
