Magic离线文档 -- 日历

日历

通过展示日历方便使用者快速填写日期，多用于需要填写日期的input表单输入控件。

日历插件

title

将年份和月份做成select控件，用户可以直接选择年份和月份

参考手册

参数配置

  • options.format : String
    输出日期的格式，默认yyyy-MM-dd
  • options.language : String
    当前语言，默认zh-CN
  • options.popupOptions : Object
    弹出层的配置项
  • options.popupOptions.hideOnEscape : Boolean
    在用户按[ESC]键时是否隐藏当前弹出层，默认true
  • options.popupOptions.offsetX : Number
    定位时的偏移量，X方向，默认0
  • options.popupOptions.offsetY : Number
    定位时的偏移量，Y方向，默认0
  • options.calendarOptions : Object
    日历的配置项
  • options.calendarOptions.weekStart : String
    定义一周的第一天取值:'Mon'|'Tue'|'Web'|'Thu'|'Fri'|'Sat'|'Sun'，默认'Sun'
  • options.calendarOptions.initDate : Date
    初始化日历的日期，默认为new Date()
  • options.calendarOptions.highlightDates : Array
    需要高亮的日期或者日期区间，格式:[date, {start:date, end:date}, date, date...]，默认[]
  • options.calendarOptions.disabledDates : Array
    不可用的日期或者日期区间，格式:[date, {start:date, end:date}, date, date...]，默认[]
  • options.title.enable : Boolean
    插件开关，默认 true

组件方法

  • show

    显示日历

  • hide

    隐藏日历

  • getDate

    获取当前日期

  • getSelectedDate

    获取当前选中日期

  • getElement

    获得DatePicker组件结构里的 HtmlElement对象

    • 参数

      • name: {String}

        可选的值包括：calendar(calendar节点)

    • 返回值: {HtmlElement}

      得到的 HtmlElement 对象

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合

组件事件

  • titlerender

    年份和月份select控件渲染完成后触发

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

  • onselectdate

    选中某个日期时触发

    • 参数

      • options: {Object}

        自定义事件参数

      • options.date: {Date}

        选中的日期

  • onshow

    日历渲染完成

  • onhide

    隐藏日历
