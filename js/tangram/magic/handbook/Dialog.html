对话框

飘浮在页面主体内容上的窗口，通常包含标题和内容两部分，是人机交互的一种特定表现方式，能起到约束用户操作范围和警示等作用。可以选择在生成对话框时同时生成遮住页面全部内容的灰度层，对话框可以多层嵌套。

对话框插件

button

对话框按钮插件

mask

对话框遮罩插件

参考手册

参数配置

  • options.titleText : Boolean
    对话框的标题内容
  • options.content : HTMLElement|String
    对话框的内容，可以是 html 或 dom 对象
  • options.contentType : String
    内容类型，可以是 element|html|text|frame，默认html
  • options.width : Number
    Dialog 的宽度，默认400
  • options.height : Number
    Dialog 的高度，默认300
  • options.left : Number
    Dialog 的左边距，默认0
  • options.top : Number
    Dialog 的上边距，默认0
  • options.draggable : Boolean
    Dialog 是否可以被拖动，默认 true
  • options.mask.enable : Boolean
    插件开关，默认false
  • options.mask.bgColor : String
    遮罩的颜色，默认#000
  • options.mask.opacity : Number
    遮罩的不透明度，默认0.15
  • options.buttons.items.text : String
    按钮名称，默认button
  • options.buttons.items.disabled : Boolean
    按钮是否失效，默认false
  • options.buttons.items.click : Function
    按钮点击click事件，默认null
  • options.buttons.items.builder : Function
    按钮构造，默认null，该属性会读取其他button属性
  • options.buttons.align : String
    按钮整体布局，默认right,可选值为left,center,right
  • options.buttons.enable : String
    插件开关，默认false

组件方法

  • render

    渲染对话框

    • 参数

      • el: {HTMLElement|id|dom}

        渲染目标容器，如果缺省，则渲染到 body 尾部

  • _renderHeader

    窗口头部构建,内部方法调用

  • isShowing

    查询对话框是否处于显示状态

    • 返回值: {Boolean}

      对话框是否处于显示状态

  • show

    显示对话框

    • 返回值: {This}

      实例本身

  • hide

    隐藏对话框

    • 返回值: {This}

      实例本身

  • setTitleText

    设置对话框标题

    • 参数

      • title: {String}

        对话框标题文本内容

    • 返回值: {This}

      实例本身

  • setContent

    设置对话框内容

    • 参数

      • content: {HTMLElement|id|dom}

        用于做为对话框内容的节点或字符串 id

      • contentType: {string}

        内容类型，可选参数有 element|html|text|frame，分别表示传入的内容类型为 dom 对象、html 字符、文本或 iframe 地址，content 参数的数据类型由 contentType 决定，两个参数配合使用，contentType 参数如果不传或非以上四种情况，一律当 html 处理

    • 返回值: {This}

      实例本身

  • focus

    聚焦对话框

  • setSize

    设置对话框尺寸

    • 参数

      • size: {Object}

        尺寸描述对象，必须至少有 width/height 中的一个

      • size.width: {Number}

        对话框的宽，单位 px

      • size.height: {Number}

        对话框的高，单位 px

    • 返回值: {This}

      实例本身

  • getSize

    获取对话框尺寸

    • 返回值: {Object}

      返回值形如 { width: 100, height: 100 }，单位 px

  • setPosition

    设置对话框的位置

    • 参数

      • pos: {Object}

        位置描述对象，必须至少有 left/top 中的一个

      • pos.left: {Number}

        对话框左边框到 body 左侧的距离，单位 px

      • pos.top: {Number}

        对话框上边框到 body 上侧的距离，单位 px

    • 返回值: {This}

      实例本身

  • getPosition

    获取对话框的位置

    • 返回值: {Object}

      返回值形如 { left: 100, top: 100 }，单位 px

  • center

    使对话框居中于浏览器视口显示

  • _createButton

    创建按钮,内部方法调用

  • renderMask

    渲染遮罩层（如果开启 mask 选项，主程序会自动渲染遮罩）

    • 返回值: {This}

      实例本身

  • showMask

    显示遮罩层（如果开启 mask 选项，主程序会自动关联遮罩显隐）

    • 返回值: {This}

      实例本身

  • hideMask

    隐藏遮罩层（如果开启 mask 选项，主程序会自动关联遮罩显隐）

    • 返回值: {This}

      实例本身

  • setWidth

    通用设置宽度

    • 参数

      • width: {Number}

        宽度值:30%|30px|30em|3cm

  • setHeight

    通用设置高度

    • 参数

      • height: {Number}

        高度值:30%|30px|30em|3cm

  • getElement

    取得 ui 模块对应的 dom element 对象

    • 参数

      • key: {String}

        该ID对应的关键字(可选参数)

    • 返回值: {Object}

      ui模块对应的dom element

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合

组件事件

  • onload

    当窗口节点渲染完成后触发

  • onbeforeshow

    当即将显示窗口时触发，如果事件回调函数返回值为 false，则阻止显示窗口

  • onshow

    当窗口显示后触发

  • onbeforehide

    当即将关闭窗口时触发，如果事件回调函数返回值为 false，则阻止关闭窗口

  • onhide

    当关闭窗口时触发

  • onfocus

    当窗口获得焦点时触发

  • onresize

    当窗口发生尺寸修改时触发

    • 参数

      • evt: {Object}

        事件参数

      • evt.width: {Number}

        宽度

      • evt.height: {Number}

        高度

  • onmove

    当窗口发生位置移动时触发

    • 参数

      • evt: {Object}

        事件参数

      • evt.left: {Number}

        左边距

      • evt.top: {Number}

        右边距
