分页

多张图片并列展示，可以采用横排或者竖排的方式，通过两边的按钮切换选中的图片，支持自动滚动和循环滚动，多用于新闻图片轮播和图片产品的图片轮播展示。

参数配置

  • options.currentPage : Number
    当前页，默认1
  • options.totalPage : Number
    总页数，默认1
  • options.viewSize : Number
    页码数，默认显示多少个页面的链接（不包括“首页”等特殊链接），默认10
  • options.currentPagePos : Number
    当前页位置，当前页面链接在页面链接列表中的默认位置，必须小于页码数，默认4
  • options.labelFirst : String
    首页链接显示的内容，默认“首页”
  • options.labelPrev : String
    上一页链接显示的内容，默认“上一页”
  • options.labelNext : String
    下一页链接显示的内容，默认“下一页”
  • options.labelLast : String
    尾页链接显示的内容，默认“尾页”
  • options.tplURL : String
    页码链接中href属性的模版，页码的模版变量为#{pageNum}，默认“##{pageNum}”
  • options.tplLabelNormal : String
    普通页码显示模版，页码的模版变量为#{pageNum}，默认#{pageNum}
  • options.tplLabelCurrent : String
    当前页码的显示模版，页码的模版变量为#{pageNum}，默认#{pageNum}

组件方法

  • render

    将分页组件渲染到dom中

    • 参数

      • target: {String|HTMLElement}

        渲染的容器，默认为document.body。

  • getElement

    取得 ui 模块对应的 dom element 对象

    • 参数

      • key: {String}

        该ID对应的关键字(可选参数)

    • 返回值: {Object}

      ui模块对应的dom element

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合

组件事件

  • onpagechange

    页码变换后触发

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

      • evt.returnValue: {Boolean}

        返回false时，会阻止<a>的浏览器默认href跳转。

      • evt.pageNum: {Number}

        变化后的页码。

  • onload

    渲染后触发

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

  • ondispose

    分页组件析构后触发

    • 参数

      • evt: {baidu.lang.Event}

        事件参数
