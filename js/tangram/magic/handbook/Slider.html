滑动条

由一个进度条和一个滑块组成，滑块可在进度条上拖动，多用于播放器展示播放进度，可以设置滑动条的刻度、滑动范围。

滑动条插件

cache

为滑动条组件增加缓存进度条

fx

为滑动条组件增加动画滚动功能

参考手册

参数配置

  • options.orientation : String
    决定滑动条是水平还是垂直，'horizontal' || 'vertical'，默认vertical
  • options.direction : String
    决定从哪一端开始移动，'forward'或'backward'，默认backward
  • options.accuracy : Float
    精确度，0-1之间的小数，滑动条滑动时会按精确度匹配位置，默认0
  • options.currentValue : Number
    滑动条的初始值，即游标初始位置，默认0
  • options.fx.enable : Boolean
    插件开关， 默认false
  • options.fx.duration : Boolean
    动画持续时间(毫秒)，默认200
  • options.cache.enable : Boolean
    插件开关，默认false

组件方法

  • render

    渲染Slider

    • 参数

      • el: {HtmlElement}

        存放slider的容器对象

  • disable

    禁用组件

  • enable

    启用组件

  • setValue

    设置组件的值

    • 参数

      • value: {float}

        要设置的值

      • noFx: {boolean}

        不使用动画，true || false 默认会使用

  • getValue

    获取组件的值

    • 返回值: {float}

      value 组件当前值

  • setRange

    设置范围

    • 参数

      • value: {float}

        设置组件的取值范围(0-1)

  • setCache

    设置缓存

    • 参数

      • value: {float}

        要设置的值

  • show

    通用展现方法

  • hide

    通用隐藏方法

  • setWidth

    通用设置宽度

    • 参数

      • width: {Number}

        宽度值:30%|30px|30em|3cm

  • setHeight

    通用设置高度

    • 参数

      • height: {Number}

        高度值:30%|30px|30em|3cm

  • setSize

    通用设置大小

    • 参数

      • size: {Number}

        {width, height}|[width, height]

      • size.width: {Number}

        宽度值:30%|30px|30em|3cm

      • size.height: {Number}

        高度值:30%|30px|30em|3cm

  • getElement

    取得 ui 模块对应的 dom element 对象

    • 参数

      • key: {String}

        该ID对应的关键字(可选参数)

    • 返回值: {Object}

      ui模块对应的dom element

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合
