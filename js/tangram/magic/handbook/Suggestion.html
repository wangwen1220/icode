输入框提示

根据输入框的内容异步获取与该内容的有关的信息，并在其下通过弹出层的方式展示，多用于搜索引擎的搜索框，支持跨域异步获取数据。

参考手册

参数配置

  • options.offset : Object
    suggestion相对于输入框的偏移量，传入的参数中可包括offsetX、 offsetY、width三个值（在CSS中使用margin同样可以定位）。
  • options.getData : Function
    在需要获取数据的时候会调用此函数来获取数据，传入的参数query是用户在输入框中输入的数据。在获取到数据后，需要触发ongetdata事件，并传入参数，例如me.fire("ongetdata", query, returnValue);
  • options.prependHTML : String
    写在下拉框列表前面的html
  • options.appendHTML : String
    写在下拉框列表后面的html
  • options.holdHighLight : Boolean
    鼠标移出待选项区域后，是否保持条目的高亮状态，默认false

组件方法

  • render

    创建输入框提示组件容器，并渲染到dom树中

  • isShowing

    判断输入框提示是否显示

  • show

    显示输入框提示容器

  • hide

    隐藏输入框提示容器

  • getInputValue

    取得input中的值

    • 返回值: {String}

      value input中的值

  • getDataByIndex

    根据index获取对应的输入框提示值

    • 参数

      • index: {Integer}

        索引

    • 返回值: {String}

      data 该索引对应的suggestion值

  • getData

    取输入框提示数据

    • 参数

      • query: {String}

        搜索关键字

  • getElement

    获得输入框提示组件结构里的 HtmlElement对象

    • 参数

      • name: {String}

        可选的值包括：input(输入框)|suggestion(suggestion部分的容器)

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合
