Magic离线文档 -- 选项卡

选项卡

通过选中不同的标题切换相应的内容，多用于功能相近的区域展示，如点卡充值/手机充值等，支持鼠标悬停或者点击两种方式切换。

选项卡插件

参考手册

参数配置

  • options.items : Array
    数据项，格式如：[{title: 'text-0', content: 'content-0'}, {title: 'text-1', content: 'content-1'}...]
  • options.selectEvent : String
    触发选项卡切换的事件名称,取值click或mouseover，默认click
  • options.selectDelay : Number
    当selectEvent是mouseover时，选项卡之间切换的延迟时间，以毫秒为单位，默认0
  • options.originalIndex : Number
    默认选项卡的打开项，默认0

组件方法

  • render

    将Tab选择卡渲染到指定的容器中

    • 参数

      • target: {String|HTMLElement}

        一个用来渲染组件的容器对象.

  • select

    切换到某个选项卡

    • 参数

      • index: {Number}

        选项卡的索引，初始值是0.

  • getCurrentTitle

    获得当前选项卡的标题

    • 返回值: {HTMLElement}

      取得当前选中项的标题容器.

  • getCurrentContent

    获得当前选项卡的内容

    • 返回值: {HTMLElement}

      取得当前选中项的内容容器.

  • getElement

    取得 ui 模块对应的 dom element 对象

    • 参数

      • key: {String}

        该ID对应的关键字(可选参数)

    • 返回值: {Object}

      ui模块对应的dom element

  • getElements

    取得所有 ui 模块对应的 dom element 对象

    • 返回值: {Object}

      取得对象的集合

组件事件

  • onbeforeselect

    当焦点发生改变之前触发该事件

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

      • evt.index: {Number}

        切换前的焦点索引

  • onselect

    当焦点发生改变时触发该事件

    • 参数

      • evt: {baidu.lang.Event}

        事件参数

      • evt.index: {Number}

        切换后的焦点索引
